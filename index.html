How to Connect (Vercel Dashboard)

Go to vercel.com → your sync-hub project → Settings → Environment Variables and add:

STRIPE_SECRET_KEY = sk_live_...
GMAIL_CLIENT_ID = ...apps.googleusercontent.com
GMAIL_CLIENT_SECRET = GOCSPX-...
GMAIL_REFRESH_TOKEN = 1//0...
TWILIO_SID = AC...
TWILIO_TOKEN = ...
TWILIO_NUMBER = +44...
LEE_PHONE = +44...

After adding keys, redeploy from Vercel dashboard (or push any commit). Status badges above will update automatically.
